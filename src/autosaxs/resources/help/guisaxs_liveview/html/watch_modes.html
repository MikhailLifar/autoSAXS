Flat vs tree mode

Switch via File → Advanced… → Switch to flat directory or File → Advanced… → Switch to tree directory (only one is shown at a time). Cannot switch while a skill is running.

ModeDiscoveryTypical outputs
FlatTop-level TIFFs in watchdir onlyUnder watchdir: averaged/, subtracted/, skill folders
TreeRecursive TIFF discoveryOutputs beside each TIFF in the tree
Session settings (calibration, buffer, analysis configs) are stored per watchdir in .guisaxs_liveview/session.yaml.
