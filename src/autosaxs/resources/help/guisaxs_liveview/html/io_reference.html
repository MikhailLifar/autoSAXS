Input and output paths

All paths below are under your Working dir unless noted.

Inputs

  • *.tif / *.tiff — new detector images
  • Calibration files — chosen in wizard (original paths unchanged)
  • Buffer .dat — chosen in wizard

Outputs

  • runs/latest/ — request.yml, result.yml, stdout.log, stderr.log
  • calibration/ — calibrate results
  • averaged_proxy/ — proxy integrate (uncalibrated)
  • averaged/ — integrate (calibrated)
  • subtracted/ — subtracted 1D curves
  • fit_distances/<stem>/, dammif/<stem>/, model_bodies/<stem>/, fit_sizes/<stem>/, mixture/<stem>/ — analysis per TIFF stem

Shared wizard configs (e.g. fit_distances.conf) sit next to per-stem folders.
