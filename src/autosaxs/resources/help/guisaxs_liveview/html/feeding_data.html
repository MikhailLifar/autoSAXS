Feeding images

Drag and drop

Drop one or more .tif / .tiff files onto the middle 2D panel. Files outside the working directory are copied into it, then queued.

Copy into watchdir

Save or copy TIFFs directly into the watch folder. In flat mode only top-level files are discovered; in tree mode nested TIFFs are found recursively (see Flat vs tree mode).

What gets queued

  • Files that appear after the app started (plus moves into the folder and overwrites of existing names).
  • TIFFs already present at launch are not auto-queued until replaced or drag-dropped.

Processing is sequential (FIFO queue).
