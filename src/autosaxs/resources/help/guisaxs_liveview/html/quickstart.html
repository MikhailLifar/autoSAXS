Quick start

  1. Activate a Python environment with autosaxs[gui] installed.
  2. Run guisaxs-liveview from the folder you want to watch (or pick one via File → Open watch directory…).
  3. Feed detector images:
    • Drag-and-drop .tif / .tiff onto the middle 2D panel (multiple files at once), or
    • Copy/save TIFFs into the watch directory.
  4. Watch the middle column for queue status and the latest 2D image.
  5. If needed: left column → Set calibration → run the wizard → Run.
  6. If needed: Set buffer → choose buffer .dat and q-range → Apply.
  7. Optional: right column → Analysis drop-down → configure wizards → new TIFFs are analyzed automatically.
