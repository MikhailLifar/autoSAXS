Calibration

Left column → Set calibration opens the wizard.

Required steps

  1. Choose a calibrant image (TIFF).
  2. Set the config file.
  3. Set the mask (use the mask wizard if needed).
  4. Press Run and wait until calibration finishes.

On success, integration uses calibrated geometry. Results go to Watchdir/calibration/. The integrator directory is saved in the session file.

Use Reset calibration in the left panel to return to uncalibrated (proxy) integration.
