Window tour

Menu bar

  • File — open working directory, Advanced… (flat/tree), exit.
  • Update — upgrade autosaxs[gui] from git main.
  • Help — this guide and About dialog.

Working dir line

Below the menu bar: contracted path to the active working directory (tooltip shows the full path).

Left column

Calibration and buffer setup wizards, previews, and reset actions.

Middle column

Live queue status, latest 2D image (drop TIFFs here), 1D curves or subtraction plots, and session history controls (< > Process).

Right column

Icon buttons to open monodisperse / polydisperse analysis windows (armed while open), and a live log with Full (skill + app) and App tabs.
