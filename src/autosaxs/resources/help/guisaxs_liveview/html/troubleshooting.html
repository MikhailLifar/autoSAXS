Troubleshooting

Skill failed

Check WorkingDir/runs/latest/stderr.log and result.yml. The UI shows a dialog for calibration and analysis skill failures.

Cannot change watchdir or watch mode

Wait until the running skill finishes.

Plots empty when browsing history

That TIFF may not have been processed with the current calibration/buffer/analysis settings. Use Process to re-run the pipeline.

Update failed

Ensure network access to the git server and write permission to the Python environment. Editable installs may need a manual git pull and reinstall.

Help missing or empty

Reinstall autosaxs[gui] or verify that autosaxs/resources/help/guisaxs_liveview/ is present in your installation (manifest.yaml, html/, style/).
