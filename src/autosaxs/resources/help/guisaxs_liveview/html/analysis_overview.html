Analysis overview

Right column → icon toolbar (monodisperse / polydisperse) and a live log (Full / App tabs).

Opening an analysis window arms that processing chain for new TIFFs while the window stays open. Closing the window disarms that chain. Both windows may be open at once; each runs its own full pipeline (including a separate Guinier fit and output tree).

With neither window open, only integration (+ subtraction if enabled) runs.

WindowSkillsMain outputs
Monodispersefit_guinier → fit_distances (+ optional shape)Window: Guinier, GNOM p(r), BODIES/DAMMIF; disk: guinier_mono/, fit_distances/, …
Polydispersefit_guinier → fit_sizes (+ optional model_mixture)Window: Guinier, D(R), optional mixture; disk: guinier_poly/, fit_sizes/, …

Toolbar buttons open or focus the window (hover for tooltips). Setup is allowed before calibration; auto analysis steps are appended only once a calibrated profile exists.
