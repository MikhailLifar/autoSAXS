Buffer subtraction

Left column → Set buffer when you need buffer-subtracted 1D curves.

Required steps

  1. Choose a buffer .dat file.
  2. Set q_min and q_max — the q-range used to match tails and estimate the buffer scale (not necessarily a wide range).
  3. Press Apply.

After buffer is configured, each new TIFF is integrated then subtracted automatically. Outputs: Watchdir/subtracted/.

In states C/CD the middle column shows sample+buffer and subtracted plots. You can open the subtraction wizard from the middle panel to adjust scaling for the current file and re-run.
