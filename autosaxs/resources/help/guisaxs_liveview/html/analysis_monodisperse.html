Monodisperse analysis

Open the monodisperse analysis wizard from the right-column icon (or focus it if already open). While the window is open, new TIFFs run Guinier → GNOM → optional shape automatically. Selecting BODIES or DAMMIF is configuration for automatic TIFF processing — it does not pause the queue. Guinier artifacts are stored under guinier_mono/ (separate from polydisperse).

Guinier

fit_guinier. Interval edits pause auto-processing and re-run Guinier then GNOM for the current profile (shape is not chained from this manual edit).

p(r)

fit_distances (AUTORG + GNOM). Wizard options: fit_distances/fit_distances.conf. Per-TIFF results: fit_distances/<stem>/ — fit PNG, p(r) plot, GNOM .out. Editing GNOM parameters re-runs GNOM only; use Re-run shape if you also need DAMMIF/BODIES again.

DAM

When DAMMIF is selected, automatic TIFF jobs run Guinier → GNOM → fit_dammif (FAST) on the GNOM output. If you select DAMMIF while a TIFF job is already mid-run (Guinier/GNOM), a shape follow-up job runs after that job finishes. Additional folder: dammif/<stem>/ — bead model, CIF, 3D preview.

Primitives

When BODIES is selected, automatic TIFF jobs include fit_bodies after GNOM. Shape list: fit_bodies/fit_bodies.conf. Per-TIFF: fit_bodies/<stem>/ — CSV, fit PNGs, 3D for best shape.
