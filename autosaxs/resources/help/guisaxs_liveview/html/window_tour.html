Window tour

Menu bar

  • File — open watch directory, switch flat/tree mode, exit.
  • Update — upgrade autosaxs[gui] from git main.
  • Help — this guide and About dialog.

Watchdir line

Below the menu bar: contracted path to the active watch folder (tooltip shows the full path).

Left column

Calibration and buffer setup wizards, previews, and reset actions.

Middle column

Live queue status, latest 2D image (drop TIFFs here), 1D curves or subtraction plots, and session history controls (< > Process).

Right column

Analysis mode drop-down, fit wizards, and result plots (including 3D where applicable).
